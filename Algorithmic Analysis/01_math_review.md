# Math Review

## Exponents
#### Definition
$$ n^a = n \times n \times n \times \dots \times n $$

#### Example
$$3^4 = 3 \times 3 \times 3 \times 3 = 81$$

#### Special Cases
- $n^1 = n$
- $n^0 = 1$
- $n^{-1} = \frac{1}{n}$
- $n^{-x} = \frac{1}{n^x}$

#### Exponent Rules
- **Multiplication:** $n^a \times n^b = n^{a+b}$  &emsp; Ex: $2^2 \times 2^3 = 2^{2+3} = 2^5 = 32$
- **Division:** $n^a / n^b = n^{a-b}$ &emsp; Ex: $3^4 / 3^3 = 3^{4-3} = 3$

## Logarithms
#### Definition
A logarithm is the inverse of exponentiation. It answers the question: 
> "To what power must I raise the base in order to produce a given number?"

If $n^a = b$, then the logarithm of $b$ with respect to base $n$ is $a$, written as

$$log_n b = a$$

#### Example
$$log_2 8 = 3 \rightarrow 2^3 = 8$$


#### Logarithm Rules

- **Product Rule:** $log(ab) = log(a) + log(b)$
- **Quotient Rule:** $log(a/b) = log(a) - log(b)$
- **Power Rule:** $log(a^b) = b log(a)$
- **Reciprocal Rule:** $log(1/a) = -log(a)$
- **Log of One:** $log(1) = 0$
